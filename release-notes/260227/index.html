    🚀 MeshMapper Update (February 27, 2026)¶

    ✨ New Features¶

    • AI MeshMapper Bot — The AI-powered MeshMapper assistant is now public! Have a question? Start a chat from the help icon on any regional map, or tag @MeshMapper in the MeshMapper Discord. The bot is continuously learning and improving (and knows only about Meshmapper).

    • Profiles & Awards — Companion/user profiles and an awards system have been added to MeshMapper! View any profile by clicking a name on a leaderboard or using the "Profiles" search button. Awards are granted automatically for milestones (data points, grid squares, leaderboard rank) or manually by administrators. See the full details and what awards are available to be earned on the Profiles & Awards wiki page.

    • Pending Repeater Approval — Regional administrators can now enable a flag that has new repeaters enter in a "Pending" state, needing approval by a regional administrator before appearing on the map. Pending repeaters have 3x the regions Stale Repeater Timer before being automatically activated, or are deleted if no advert is received within 1x the regions Stale Repeater Timer. A new Discord notification type has been created that will notify regional administrators once a day if the region has any pending repeaters. This is an anti-abuse measure and should be used with caution, as it can cause inaccurate map data. Read about it more here.

    • New Administrators — Regional administrators can now create new regional administrator accounts. Create the account and send the new administrator the registration link to complete their registration. Read about it more here.

    • Top Explorers (Regional) — The Top Explorers leaderboard, which awards a point for being the first to wardrive a grid square, has been added to regional leaderboards. (Thanks ded)

    • Global Leaderboard Additions — The global leaderboard now includes rankings for Most Repeaters, Most Grid Squares, Most Contributors, and Most Data Points across all regions.

    🐛 Bug Fixes¶

    • Fixed an issue where the map legend was layered above the help button. (Thanks Alex The Emperor Penguin)
