# Report Bugs & Request Features

MeshMapper is actively being developed using community feedback, and bugs are expected.

You can submit a ticket easily using our guided form:

<a href="https://meshmapper.net?createticket" target="_blank" style="background-color: #007bff; color: white; padding: 10px 20px; text-decoration: none; border-radius: 5px; font-weight: bold; display: inline-block; margin: 10px 0;">Submit Bug or Feature Request</a>

Alternatively, you can report bugs directly on the MeshMapper Central Issue Tracking and Project Planning Github repo here:

<https://github.com/MeshMapper/MeshMapper_Project>

We also welcome ideas and suggestions to improve MeshMapper.  MeshMapper is a community-driven product and all feedback is welcome.
