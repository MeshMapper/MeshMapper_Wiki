# System Status

!!! note
    The live status dashboard that used to be embedded on this page has been retired.

## Checking MeshMapper Status

- **Discord** — Outages and scheduled maintenance are announced in the [MeshMapper Discord server](https://discord.gg/tyXbecdxgr). This is the fastest place to check whether an issue is already known.
- **Maintenance mode** — When MeshMapper is undergoing maintenance, the wardriving app displays a maintenance message with details. You can continue wardriving in [Offline Mode](app_connection_guide.md#offline-mode) and upload your sessions once service is restored.
- **Your region's ingestion** — If the platform is up but your region's data seems stale, check **Region Info → Observers** on your region's map to see whether your region's MQTT observers are online, or ask the Discord bot: `@MeshMapper !status YOW`.

If you're seeing a problem that isn't announced anywhere, please report it — see [Report Bugs & Features](reportbugs.md).
