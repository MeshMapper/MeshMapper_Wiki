# 🚀 MeshMapper Update (March 6, 2026)

### ✨ New Features

* **Settings Panel** — A new settings panel has been added to the map interface, providing quick access to display preferences, coverage options, and grid transparency controls all in one place.

![Image](https://meshmapper.net/img/upload_20260306_173533_55aeb5b0.jpg)

* **Adjustable Grid Transparency** — Users can now adjust the opacity of coverage grid squares directly from the settings panel, with separate sliders for normal and faded opacity levels. *(Thanks davidjay)*

![Image](https://meshmapper.net/img/upload_20260306_100217_2cd63976.jpg)

* **Effective Coverage Color Spectrum** — The effective coverage layer now supports a full-spectrum colour mode (Red → Blue) in addition to the default Red → Green, selectable from the settings panel. *(Thanks HerculesMulligan)*

![Image](https://meshmapper.net/img/upload_20260306_100253_9265c974.jpg)

* **Active Wardriving Indicators** — The main landing page now shows regions with active wardriving sessions as pulsing green icons on the map, making it easy to see where wardrivers are currently collecting data in real time.

![Image](https://meshmapper.net/img/upload_20260306_100327_62c3e436.jpg)

* **Consider Failed DISC as DROP** — A new region-level setting allows administrators to treat failed discovery (DISC) pings as DROP pings. This functionality is coming to the mobile apps shortly.

![Image](https://meshmapper.net/img/upload_20260306_100419_92939b5f.jpg)

* **Bulk Delete on Coverage Table** — The coverage data table in the admin panel now supports multi-select with bulk delete. *(Thanks davidjay)*

* **Hide Dissociated Pings** — A new option in the map settings will prevent pings associated with missing/old repeaters from displaying on the map. *(Thanks Vjo)*

![Image](https://meshmapper.net/img/upload_20260306_122758_d6423708.jpg)

* **Effective Coverage Minimum Sample Size** — An option has been added to the map settings that allows for configuring the minimum number of samples/pings required in order to place an effective coverage grid square on the map. *(Thanks AI7NC)*

![Image](https://meshmapper.net/img/upload_20260306_171313_48c3b06d.jpg)

* **Selectable Grid Size** — Grid square sizes can now be adjusted in the map settings! *(Thanks tkelemen)*

![Image](https://meshmapper.net/img/upload_20260306_173613_e606309c.jpg)


### 🐛 Bug Fixes

* Fixed an issue where spaces/whitespace was allowed within the scopes field, causing an error on the app.

* Fixed an issue where coverage pings could not be opened on the map from the admin panel when in certain coverage display modes. *(Thanks NoahK)*

* Fixed double-encoding issue on Repeaters table in admin panel. *(Thanks NoahK)*
